# API Cep.App

API para consulta de dados de endereço do Brasil contendo endereço completo e coordenadas de latitude e longitude.

## Projetos:
		
###	[Api]
	Cep.App.Api: Web Api para consulta de endereço a partir do CEP;

## Requisitos

### .NET Core SDK >= 6.0

Esta API depende do [.NET Core SDK](https://dotnet.microsoft.com/download/dotnet/6.0), com versão igual ou superior ao 6.0.

### Visual Studio 2022
### SQL Server Express 2019
